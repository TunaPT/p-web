MineSweeper - Fundamentos de Programação - 2021

MineSweeper is game which to win the player must discover all empty places, without selecting a mine. If a square that contains a mine is pressed, the game is over and the player loses. This projects aims to developed this game with a console based system, where the player can define how many lines, columns and mines the game will have. It's useful to adapt the difficulty - Repository

DeisiFlix - Algoritmia e Estrutura de Dados - 2022

This project aims to develop an application in Java language thatthat is able to receive and respond to requests for statistics about a database of films. Taking into account that one of the issues addressed in AED is the question of the efficiency of of data structures and algorithms, it is intended that the program developed is not only functionally correct, but also to be as fast as possible - Repository

DeisiJungle - Linguagens de Programação II - 2022

One day, in a savannah very (very) far from here, the animals decided to find out who would be the best athlete amongst them. To do so, they defined a set of sporting events that they would compete against each other - whoever won the most events would be considered the King of the Jungle. The news quickly reached Tarzan, who did not want to stop participating in the competitions, convinced that he would easily win.The first competition is the athletics race. The animals will gather on a track and compete to determine which is the best athlete - Repository
